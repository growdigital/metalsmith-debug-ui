Metalsmith build

simple ms procedure to build the client I couldn't get webpack to work here so I'm doing that externally.

 
Metalsmith(baseDir)
.source(resolve(baseDir, 'lib', 'client'))
.destination(resolve(baseDir, 'dist', 'client'))
.use(less({
  pattern: 'styles.less'
}))
.use(ignore([
  'styles.less'
]))
.use(inPlace())
.build((err) => {
  if (err) dbg(err)
  else dbg('done')
})
