Metalsmith build

simple ms procedure to build the client

 
Metalsmith('./')
.source('lib/client')
.destination('dist/client')
.use(webpack())
.use(move({
  '**/*.jsx': false
}))
.use(less({
  pattern: 'styles.less'
}))
.use(ignore([
  'styles.less'
]))
.use(inPlace())
.build((err) => {
  if (err) dbg(err)
  else dbg('done')
})
